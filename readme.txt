== Description ==

You should be an Experian CheetahMail Fr customer for using this plugin.

With this plugin, you can:

* Manage your emailing campaigns
* Manage your newsletters: number of elements, frequency, ...
* Customize your email:  look, envelopes (header and footer)
* Create target to send your emailing
* Manage the data exchange between WordPress and Cheetahmail
* Preview of your next send
* Stats about your sends

You can set the following options:

* Data synchronize between WordPress and CheetahMail
* Envelopes of your email
* Synchronize subscribers
* Content of your email

Prerequisites

* You have to be a client of CheetahMail FR
* You have to enable API on your account especially Email Services package composed by bodymanager, campaigns, chronocontact, configs, filters, subscribers, stats

== Installation ==

1. Download
2. Upload to your `/wp-contents/plugins/` directory or let wordpress do the job
3. Activate the plugin through the 'Plugins' menu in WordPress.


== Uninstallation and plugin deletion ==

1. Deactivate the plugin through the 'Plugins' menu in WordPress.
2. Delete the plugin through the 'Plugins' menu in WordPress. Notice that, the database is not deleted.
